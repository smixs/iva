## Что это

Iva - минималистичный личный AI-агент с долговременной памятью на [eve](https://eve.dev/docs/introduction), агентном фреймворке от Vercel. Живёт в Telegram, крутится на вашем сервере. Ставится одной командой, настраивать почти ничего не надо - всё уже выбрано за вас.

Главное здесь - память. Каждый разговор Iva разбирает на факты: о вас, людях вокруг, проектах, решениях - и складывает в дерево, которое сама же приводит в порядок каждую ночь. Чем дольше пользуетесь, тем лучше она вас знает.

И при этом Iva дешёвая. Минимум - одно ядро, 512 МБ памяти, 10 ГБ диска: дроплет DigitalOcean за $4 плюс модель через OpenCode Go за $5. Итого около $9 в месяц за агента, который пришлёт утренний дайджест и напомнит про встречу. Код и память - ваши, ключ и модель выбираете сами. Полностью open source.

## Почему ещё один агент

[OpenClaw](https://github.com/openclaw/openclaw), [Hermes](https://github.com/NousResearch/hermes-agent), [nanobot](https://github.com/HKUDS/nanobot) - выбор уже есть, и неплохой. Проблема в том, что каждый из них сваливает на вас одну и ту же кучу решений: какую модель, какую память, какой поиск, как развернуть, как это всё связать. Слишком много вариантов - это и есть настоящая боль.

Я этот выбор сделал сам. Постоянно тестирую агентов, модели, связки и харнессы, оставляю то, что реально работает, и собираю в Iva - чтобы вам не пришлось копаться в этом самостоятельно. Цель простая: дешёвый, быстрый и надёжный агент на каждый день, который сам присылает утренний дайджест.

- **Лучшее из разных агентов в одном месте.** Я этим занимаюсь давно, отбираю хорошие идеи и собираю их с простыми настройками.
- **Открыт для доработки.** Код открыт, модели открыты - DeepSeek, Kimi, GLM сейчас реально сильные, переключаетесь на лету.
- **Одна команда - и всё работает.** Telegram для чата, Deepgram для голоса, древовидная память, ночные сводки - уже собрано и настроено. Что-то вроде Linux Mint среди AI-агентов.

Агентов вокруг много. Этот - мой. Теперь и ваш.

## Память - то, что копится

Большинство агентов забывают вас в тот момент, когда переполняется контекстное окно. Iva - нет. Её память устроена как дерево. Ива - дерево, тут всё логично.

```
        🪵  СТВОЛ    - год + карточки людей, проектов, решений (общая картина)
       ╱  ╲
      🌿 ВЕТВИ      - месячные выжимки, собранные из недель, собранных из дней
     ╱      ╲
    🍃 ЛИСТЬЯ        - полный, дословный транскрипт каждого дня
```

- Листья - сырой транскрипт каждого дня, дословно.
- Ветви - короткие выжимки: сначала за день, потом неделя собирается из дней, месяц из недель.
- Ствол - всё сходится в общую картину: год плюс карточки фактов о людях, проектах и решениях, которые важны.

Каждую ночь Iva сама наводит порядок: из листьев делает выжимки и собирает их вверх по веткам. Поэтому она одинаково легко вспомнит дословно, что было в конкретный вторник, и расскажет, чем вы занимались весь месяц.

Это память с низким контекстом по задумке. Iva никогда не грузит в модель всю свою историю. В контексте всегда лежит один крошечный файл CORE - кто вы, постоянные предпочтения, активные цели. Всё остальное подтягивается под конкретную задачу обычным поиском по файлам.

Над памятью я работаю дольше всего: сначала [agent-second-brain](https://github.com/smixs/agent-second-brain), потом скилл типизированного графа [autograph](https://github.com/smixs/autograph/tree/main), и весь этот опыт собран здесь. Дерево выше - иерархический DAG выжимок: старые дни сжимаются, но указатель на оригинал остаётся. В основе - идея из пейпера [LCM: Lossless Context Management](https://papers.voltropy.com/LCM) (Ehrlich & Blackman, 2026) плюс моя собственная доработка сверху. Одна из лучших схем памяти, какие есть сейчас у личного агента, и работает на открытых моделях, которые принадлежат вам, без подписки.

## Провайдеры и цена

Сама Iva бесплатна, код открыт. Платите только за сервер и подписку на модель:

- Сервер - любой всегда включённый (VPS на 1-2 ГБ памяти, около $5/мес), или собственный компьютер, пока он работает.
- Модель - выбираете одного провайдера, оба совместимы с OpenAI API и работают с любого IP:
  - OpenCode Zen (Go) - около $5/мес, лимиты поскромнее. Самый дешёвый старт.
  - Ollama Cloud - около $20/мес, лимиты пощедрее.

  Внутри провайдера выбираете модель (я бы советовал DeepSeek). Наценки сверху цены провайдера нет.
- Голос - [Deepgram](https://console.deepgram.com) для расшифровки, дают бесплатный стартовый кредит.

## Установка

1. Откройте терминал на сервере (или на своём компьютере).
2. Вставьте команду и нажмите Enter.

   ```bash
   curl -fsSL https://raw.githubusercontent.com/smixs/iva/main/install.sh | bash
   ```
3. Установщик сначала спросит язык - English или русский, потом проведёт по каждому ключу с прямой ссылкой, вы просто вставляете значения. В какой-то момент попросит написать боту любое сообщение - так Iva запомнит вас и будет отвечать только вам.
4. Готово. Пишите боту в Telegram, Iva ответит.

Подробнее про запуск на VPS - [DEPLOY.md](DEPLOY.md).

> Как попасть на арендованный сервер: хостинг присылает адрес (IP), логин и пароль. На Mac или Linux откройте "Терминал", на Windows - "PowerShell", введите `ssh root@ВАШ_АДРЕС`, потом пароль. Вы внутри.

## Приватность

Код и память остаются на вашем сервере. Vault - отдельный приватный git-репозиторий: один раз привязали remote, и память бэкапит себя сама. Ключи лежат в `.env`, а не в коде. Бот отвечает только тем Telegram ID, которым вы разрешили, всем остальным молчит по умолчанию.

Честно про границу: модель и расшифровка голоса идут через облачные API - те, что вы выбрали и оплачиваете сами. Self-hosted здесь касается вашего кода и памяти, а не весов модели.

## Чего Iva не делает

Чтобы вы точно знали, что берёте.

- Только Telegram. Ни веб-приложения, ни дашборда - чат и есть весь интерфейс.
- Отвечает на языке, выбранном при установке. Переключить можно, но не одновременно на двух.
- Бэкап памяти - это `git push` в репозиторий, который вы заводите один раз, а не облачная синхронизация.
- Поиск буквальный, не семантический. Iva ищет по файлам, векторного поиска нет.
- Один пользователь. Один владелец, одна память - не командный и не мультитенантный ассистент.
- Версия до 1.0. Работает и используется каждый день, но молодая. Шероховатости будут - пишите о них.
