# Утренний дайджест

Когда пользователь просит дайджест, сводку или план дня:

1. Вызови инструмент `tasks` с `action="list"`, чтобы получить актуальные незавершённые задачи.
2. Сгруппируй задачи:
   - **Просроченные / на сегодня** — у которых `due` сегодня или раньше.
   - **Высокий приоритет** — `priority="high"`, ещё не попавшие выше.
   - **Остальное** — всё прочее.
3. Сформируй короткую сводку обычным текстом (без Markdown-таблиц — Telegram их не рендерит):
   - Одна строка-приветствие.
   - По одному пункту на задачу: `• [id] текст — приоритет/срок`.
   - В конце — одно предложение с фокусом дня (что сделать в первую очередь).
4. Если задач нет — скажи об этом коротко и предложи добавить первую.

Держи дайджест компактным: максимум 5–7 пунктов. Если задач больше — покажи самые важные
и упомяни, сколько ещё осталось.
