# Changelog

## [0.1.0] — 2026-06-20

Первый релиз. Личный ИИ-помощник с памятью в Telegram, который заводится одной командой.

- 🎙️ Голос и видео — расшифровывает речь на любом языке
- 🧠 Древовидная память (день → неделя → месяц → год) — сама наводит порядок по ночам
- 🔎 Быстрый поиск по памяти
- 🤖 Выбор модели — какой ИИ под капотом (рекомендуется DeepSeek)
- 🧩 Навыки (скиллы) и подключения через MCP
- 🎛️ Команды в Telegram: `/help` `/task` `/tasks` `/digest` `/new` `/restart`
- 🔒 Безопасность и приватность — отвечает только тебе, память хранится у тебя
- 🎭 Настраиваемая личность
