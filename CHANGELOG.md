# Changelog

## [0.1.3] - 2026-06-22

Патч: разметка в Telegram, англоязычный установщик, слабые VPS и фикс модели OpenCode.

- 💬 **Разметка Telegram везде** — единый закалённый конвертер markdown → Telegram-HTML, через который теперь идёт КАЖДОЕ сообщение: ответы в чате, ночные отчёты, утренний дайджест. Раньше cron-отчёты приходили сырым `**текстом**` с `---` и бэктиками. Конвертер не падает ни на каком вводе и всегда отдаёт валидный HTML (баланс тегов, экранирование, rich-разметка по докам Telegram); на отказ Telegram — мягкий self-heal без потери текста и без петель.
- 🌍 **Установщик на двух языках** — язык спрашивается ПЕРВЫМ вопросом (дефолт English), и весь вывод `install.sh` идёт на выбранном языке. Выбор пробрасывается в агента и vault, поэтому спрашивается один раз.
- 💾 **Авто-своп для слабых VPS** — на сервере с <1.5 ГБ RAM без свопа `eve build` падал по OOM (код 137, «Killed»). Теперь установщик сам заводит swapfile 2 ГБ перед сборкой (идемпотентно, с проверкой места). Iva ставится даже на $4 DigitalOcean droplet (512 МБ).
- 🤖 **Фикс модели OpenCode Go** — Iva слала ID с префиксом `opencode-go/`, эндпоинт отвечал «Model … is not supported». Теперь уходит чистый ID (`deepseek-v4-pro`); старые `.env` с префиксом чинятся сами после `iva update`.
- 🌳 **Дерево при обновлении** — `iva update` рисует то же ANSI-дерево, что и установка.

## [0.1.2] - 2026-06-21

Патч: надёжный запуск и веб-поиск.

- 🔌 **Свой порт** — сервер на настраиваемом `IVA_PORT` (дефолт `8723`) вместо занятого многими `3000`. Бот больше не «молчит» из-за конфликта портов; старые установки мигрируются автоматически при `iva update`.
- 🔎 **Веб-поиск с выбором провайдера** — Tavily / Exa / Parallel / Brave, выбираешь при установке (или `iva config`), один ключ на провайдера. DuckDuckGo убран — с серверного IP он отдавал капчу.
- 🩺 **Диагностика** — `iva doctor` проверяет порт и ключ активного поисковика; preflight-проверка занятости порта при настройке.
- 🧹 **Зелёный typecheck** — починен `parse_mode` в Telegram-канале.

[0.1.2]: https://github.com/smixs/iva/releases/tag/v0.1.2

## [0.1.0] - 2026-06-20

Первый релиз. Персональный AI-агент с памятью в Telegram, который заводится одной командой.

- 🎙️ Голос и видео - расшифровывает речь на любом языке
- 🧠 Древовидная память (день, неделя, месяц, год) - сам наводит порядок по ночам
- 🔎 Быстрый поиск по памяти
- 🤖 Выбор модели - какой ИИ внутри (OpenCode Go от $5/мес или Ollama Cloud, рекомендую DeepSeek)
- 🧩 Навыки и подключения через MCP
- 🎛️ Команды в Telegram: `/help` `/task` `/tasks` `/digest` `/new` `/restart`
- 🔒 Отвечает только вам, память хранится у вас
- 🎭 Характер меняется прямо в разговоре
